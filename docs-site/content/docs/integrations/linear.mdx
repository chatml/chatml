Linear integration brings issue tracking context directly into your AI coding sessions.

## Connecting Linear

1. Go to **Settings** (`Cmd+,`) > **Account** > **Linear**
2. Click **Connect Linear**
3. Authorize in your browser
4. ChatML receives the authorization via deep link

## Features

Once connected:

- **Issue discovery** — ChatML automatically detects Linear issues from branch names, CLI arguments, or recent commits
- **Issue context** — Issue details are provided to Claude via MCP tools, giving the AI full context about what you're working on
- **Issue operations** — Search, create, update, and comment on Linear issues from within ChatML

## Issue Format

Linear issues are identified by their team prefix and number, for example: `CHA-42`, `ENG-123`.

## Disconnecting

Go to **Settings > Account > Linear** and click **Disconnect** to remove the integration.
